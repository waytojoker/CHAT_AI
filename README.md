# CHAT_AI

## 环境配置

```
pip install -r requirement.txt
```



## RAG系统集成

### RAG系统启动

```
streamlit run D:\Project\Python\CHAT_AI\modules\rag_main.py
```

D:\Project\Python\CHAT_AI\modules\rag_main.py可替换为本地文件路径

### 离线模型需配置千帆令牌

目前令牌是收费的，可以自行配置，或者在实训群内索要

配置好令牌后点击测试千帆连接，连接通畅后可正常使用
